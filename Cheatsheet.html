Hoe maak je een cheatsheet:

  1. Begin met het maken van een github repository op github, dat doe je door op het + icoontje te klikken als je ingelogt bent in github. Dan maak je een nieuwe repository aan.
  2. Daarna moet je jezelf locaten in de map waar jij je github bestanden wilt uploaden, dat doe je met cd Bijvoorbeeld: Cd C:/Users/Test/Github
  3. Maak een bestand aan die je wilt uploaden naar github en voeg hem toe in de folder die je hebt aangemaakt op je computer.
  4. Doe dan “git init”
  5. Nu doe je “git add .”
  6. Vervolgens commit je de file, dit doe je met “ git commit -m “TEXT” “
  7. Dan doe je “git branch -M main”
  8. Daarna “git remote add git@github.com:(githubusername)/repositoryname).git
  9. Nu push je de files naar github met “git push -u origin main”
